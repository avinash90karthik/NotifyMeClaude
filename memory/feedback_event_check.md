**How to apply:**
1. Bei JEDER Analyse als ERSTEN Schritt: "Gibt es angekündigte Events? Sind sie SCHON passiert?"
2. Premarket-Kurse als Signal nutzen — wenn MU -14% im Premarket steht, ist ETWAS passiert
3. Nicht in US-Zeit denken — Abdullah ist in Deutschland (CET). "Heute Abend US" = "gestern Nacht CET" möglich
4. Wenn ein Event bereits stattfand: Marktreaktion analysieren BEVOR Trade-Empfehlung
